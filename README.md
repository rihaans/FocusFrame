# FocusFrame (Local MVP)

## Quickstart
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/Mac: source .venv/bin/activate
pip install -r requirements.txt

# Run demo (generates a test notification every 30s)
python -m focusframe.main --demo
```
